# Nerviq Commercial Licensing

Nerviq is open source under the **GNU Affero General Public License v3.0 (AGPL-3.0)**.

## Open Source License (AGPL-3.0)

The AGPL-3.0 license allows you to freely use, modify, and distribute Nerviq as long as derivative works and network-facing deployments also remain open source under the same license. This covers:

- Using the CLI locally or in CI pipelines
- Running audits and governance checks on your own repos
- Contributing improvements back to the project
- Internal company use where the software is not offered as a service to third parties

## When You Need a Commercial License

A commercial license is required if you want to:

- **Host Nerviq as a service** — offering Nerviq functionality to third parties over a network (SaaS, managed service, API)
- **Embed in proprietary products** — bundling Nerviq into closed-source software or a commercial SaaS platform
- **Distribute modified versions** — shipping changes to Nerviq without releasing source code under AGPL-3.0

## What's Always Free

No commercial license is needed for:

- CLI usage (local and CI/CD)
- GitHub Action integration
- VS Code extension
- Local and team-internal audits
- Internal tooling that is not offered as a service to others
- Contributing to the open source project

## FAQ

**Can I use Nerviq in my company for free?**
Yes. Using the CLI, CI integration, or VS Code extension internally is fully covered by AGPL-3.0 at no cost.

**Can I modify Nerviq for internal use?**
Yes. Internal modifications that are not distributed or offered as a network service do not trigger AGPL copyleft requirements.

**Do I need a license for my CI/CD pipeline?**
No. Running Nerviq in your own pipelines is free under AGPL-3.0.

**What if I build a product that calls Nerviq under the hood?**
If the product is proprietary or SaaS and you distribute or serve it to others, you need a commercial license.

**Can I contribute and still use a commercial license?**
Yes. Contributions to the open source project are welcome regardless of your licensing arrangement.

**What happens if I'm unsure whether I need a commercial license?**
Reach out — we're happy to help clarify. Most usage patterns are covered by the free AGPL-3.0 license.
