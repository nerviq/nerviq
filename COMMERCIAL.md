# Nerviq Licensing

Nerviq is open source under the **MIT License** (relicensed from AGPL-3.0 in
v1.31.0, July 2026).

## What this means

**Everything is free, for everyone, for any use.** The MIT license permits
use, copying, modification, merging, publishing, distribution, sublicensing,
and selling — in open-source, proprietary, internal, and SaaS contexts alike.

No commercial license is required for:

- CLI usage (local and CI/CD)
- GitHub Action integration
- VS Code extension
- SDK / MCP server integration
- Embedding Nerviq in proprietary products
- Hosting Nerviq-powered services (SaaS, managed service, API)
- Distributing modified versions

The only requirement is the standard MIT one: keep the copyright and
permission notice in copies or substantial portions of the Software.

## Why the change?

The previous AGPL-3.0 + commercial dual-license model protected revenue that
did not exist yet, at the cost of adoption friction (AGPL is on many
enterprise default-deny lists). As an adoption-stage developer tool, Nerviq
is better served by the most permissive mainstream license available.
